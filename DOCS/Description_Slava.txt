2015.08.18
	Проблема с форматом даты. Пробую использовать UI Angular компонент datepicker.
	При сериализации Json() дата превращается в "\/Date(1198908717056)\/", что
	не подходит для datepicker. Стал использовать newtonsoft JsonConvert,
	там можно задавать нужный формат даты с помощью JsonSerializerSettings используя
	свойство DateFormatString.
	Остановился на том, что дата рождения не работает. Если type="text", то из базы
	загружает, но менять можно, что угодно. Если type="date", то из базы не загружает.
	--
	Оказалось, что если type="date" то поле должно быть явно типа Date, т.е. надо
	делать birthday = new Date("some_date"). Может быть лучше использовать type="text"?
